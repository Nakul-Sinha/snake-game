# Snake Game

A simple snake game implemented using HTML, CSS, and JavaScript.

## How to Play

Use the arrow keys (Up, Down, Left, Right) to control the snake's movement. The goal is to eat the food (represented by a red square) to grow the snake. Avoid colliding with the walls or the snake's own body.

## Controls

*   **Up Arrow:** Move the snake up.
*   **Down Arrow:** Move the snake down.
*   **Left Arrow:** Move the snake left.
*   **Right Arrow:** Move the snake right.

## Game Over

The game ends if the snake collides with:

*   The walls of the game board.
*   Its own body.
